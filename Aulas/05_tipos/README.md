# Aula 05 - Tipos de Dados no Pandas

## Objetivo

Este módulo oferece duas opções temáticas à equipe responsável:

**Opção A — Limpeza, Tratamento e Transformação de Dados:**  
Apresentar os principais procedimentos de preparação de dados para análise. Ao final, o participante será capaz de identificar e tratar dados ausentes, remover inconsistências, padronizar formatos, tratar outliers e transformar variáveis.

**Opção B — Introdução ao Machine Learning:**  
Introduzir os conceitos fundamentais de Aprendizado de Máquina, com foco em Aprendizado Supervisionado. Ao final, o participante compreenderá os tipos de aprendizado, o fluxo de treinamento e avaliação de modelos, e os principais algoritmos de classificação e regressão.

## Conteúdos (Opção B — ML)

- Introdução ao Aprendizado de Máquina
- Divisão de dados: método Holdout
- Classificação: conceitos e principais algoritmos
- Regressão: conceitos e principais algoritmos
- Avaliação de modelos e métricas de desempenho

## Considerações Finais

Seja pela limpeza dos dados ou pela construção de modelos preditivos, este módulo representa a fronteira entre análise exploratória e aplicação prática da Ciência de Dados em problemas reais.
