# Aula 04 - Leitura de Arquivos (CSV e Excel)

## Objetivo

Explorar técnicas de visualização para análise e comunicação de dados. Ao final deste módulo, o participante será capaz de escolher o tipo de gráfico adequado para cada situação e produzi-lo com boas práticas visuais, tornando os resultados de uma análise compreensíveis para qualquer audiência.

## Conteúdos

- Importância da visualização na análise exploratória
- Gráfico de barras
- Gráfico de pizza
- Gráfico de dispersão
- Histogramas
- Boxplot
- Gráfico de violino
- Outros gráficos relevantes
- Boas práticas de visualização

## Considerações Finais

Uma visualização bem construída comunica em segundos o que levaria parágrafos para explicar em texto. Saber escolher e montar o gráfico certo é uma habilidade tão importante quanto saber calcular a estatística.
