# Aula 08 - Operações de Agregação

## Objetivo

Compreender como calcular estatísticas resumidas de um DataFrame e como agrupar dados para análises comparativas. Ao final deste módulo, o participante conseguirá computar médias, somas, contagens e outras métricas estatísticas segmentadas por grupos.

## Conteúdos

- Funções estatísticas básicas (`mean()`, `sum()`, `median()`, `min()`, `max()`, `count()`)
- Agrupamento de dados utilizando o método `.groupby()`
- Agregações múltiplas utilizando o método `.agg()`
- Tabelas dinâmicas básicas e contagem de valores únicos com `.value_counts()`

## Considerações Finais

O agrupamento de dados (`groupby`) é uma das ferramentas mais poderosas do Pandas. Ele permite extrair insights e compreender o comportamento de diferentes categorias contidas na base de dados de forma extremamente rápida.
