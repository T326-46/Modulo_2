# Aula 01 — Introdução ao Pandas e Dados Tabulares

## Objetivo

Apresentar os conceitos gerais da área de Ciência de Dados e os fundamentos da linguagem Python. Ao final deste módulo, o participante será capaz de compreender o que é Ciência de Dados, entender as etapas de um projeto na área e escrever programas básicos em Python utilizando variáveis, operadores, estruturas de controle e funções.

## Conteúdos

- Conceitos gerais sobre Ciência de Dados
- Etapas de um projeto de Ciência de Dados
- Introdução à linguagem Python
- Tipos básicos e variáveis
- Expressões e operadores
- Estruturas condicionais
- Estruturas de repetição
- Estruturas de dados: Listas, Tuplas, Conjuntos e Dicionários
- Definição e uso de funções

## Considerações Finais

Python se consolidou como a principal linguagem da Ciência de Dados pela sua simplicidade e ecossistema. Os fundamentos abordados aqui são a base de tudo que vem nos módulos seguintes.
