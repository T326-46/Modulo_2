# Aula 01 — Introdução ao Pandas e Dados Tabulares

## Objetivo

Apresentar os conceitos gerais da área de Ciência de Dados e Python. Ao final deste módulo, o participante será capaz de compreender quais as principais definições e conceitos em ciência de dados com python, e  o conhecimento sobre a principal ferramenta: pandas.

## Conteúdos

- Conceitos gerais sobre Dados
- Etapas de um projeto de Ciência de Dados

## Considerações Finais

Python se consolidou como a principal linguagem da Ciência de Dados pela sua simplicidade e ecossistema. Os fundamentos abordados aqui são a base de tudo que vem nos módulos seguintes.
