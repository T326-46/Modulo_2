# Aula 02 — Estruturas Fundamentais: Series

## Objetivo

Introduzir a manipulação de dados tabulares utilizando a biblioteca Pandas. Ao final deste módulo, o participante será capaz de compreender o tipo de dado que existe nas linhas e colunas de um Objeto Pandas.

## Conteúdos

- Estruturas fundamentais: Series


## Considerações Finais

A manipulação adequada dos dados é pré-requisito para qualquer análise confiável, e o conceito de Series é um entendimento fundamental de como virão as linhas e colunas nos programas com pandas.
