# Aula 06 - Seleção de Linhas e Colunas

## Objetivo

Compreender como selecionar partes específicas de um DataFrame, tais como linhas, colunas ou células individuais. Ao final deste módulo, o participante conseguirá acessar e extrair com precisão qualquer dado contido em um DataFrame.

## Conteúdos

- Seleção de colunas com a sintaxe de colchetes `df['coluna']`
- Acesso a atributos de colunas com `df.coluna`
- Seleção por rótulos (nomes de linhas e colunas) utilizando `.loc[]`
- Seleção por posições inteiras (índices numéricos) utilizando `.iloc[]`

## Considerações Finais

A seleção eficiente de subconjuntos de dados é uma das operações mais comuns em qualquer fluxo de análise, permitindo isolar a informação de interesse para manipulações futuras.
