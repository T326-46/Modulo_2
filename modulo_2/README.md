# Módulo 2 — Python para Ciência de Dados

## Objetivo

Introduzir a manipulação de dados tabulares utilizando a biblioteca Pandas. Ao final deste módulo, o participante será capaz de carregar, inspecionar, filtrar e transformar datasets reais, compreendendo como a qualidade da manipulação de dados impacta diretamente a qualidade das análises.

## Conteúdos

- Manipulação de dados tabulares com Pandas
- Estruturas fundamentais: Series e DataFrame
- Tipos de dados no Pandas
- Leitura de arquivos CSV e Excel
- Renomeação de colunas
- Seleção de linhas e colunas
- Filtros e consultas
- Operações básicas de agregação
- Tratamento inicial de dados

## Considerações Finais

A manipulação adequada dos dados é pré-requisito para qualquer análise confiável. Um dado mal tratado compromete todos os resultados subsequentes, independente da sofisticação do modelo utilizado.
